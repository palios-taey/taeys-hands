# Project: consult-platform-independence - Per-platform driver+YAML+monitor: full isolation
> P0, Jesse-directed 2026-07-03 ("everyone needs their own YAML, their own driver and their own monitor... Each platform is completely independent"). ROOT CAUSE this fixes (Observed): the shared driver base + shared monitor couple all 5 platforms — one shared-monitor change (e132bf15) broke Grok+Gemini-DR+Perplexity-DR simultaneously (MONITOR_REGRESSION_ARCHAEOLOGY_2026-07-03.md), and the audit-fix plan had to depends-chain every base.py fix because each touched all platforms. Target shape: consultation_v2/platforms/<platform>/ package holding THAT platform's driver.py + monitor.py + <platform>.yaml, importing ONLY leaf AT-SPI/runtime primitives + types — never another platform, never a shared behavioral base, never a shared monitor. Duplication across the 5 packages is ACCEPTED BY DESIGN: independence > DRY. Absorbs the parked audit-fix branches (chatgpt 599074da, claude a04da10a) so no validated fix is lost. USAGE DISCIPLINE (Jesse 2026-07-03): taeys-hands (Fable) does spec/verdicts/gates ONLY — all code by taeys-hands-codex, architecture review by FULL DCM council (dcm-council, real CLIs), and ALL consult-driving/production-validation executed by a spawned Opus 4.8 subagent or CLI peer with Fable reviewing returned evidence only.
